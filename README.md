# gts
Timestamp helper

## Install

```
go get github.com/myyang/gts
```

## Usage

```
Usage of gts:
  -dt string
        UTC datetime string to parse. Format: RFC3339 (default "1970-01-01T00:00:00Z")
  -p string
        Timestamp precision, default: 'ms'. Options: 's', 'ms', 'us', 'ns' (default "ms")
```
